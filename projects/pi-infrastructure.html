Docker · Linux · Claude Code

Self-Hosted Infrastructure

A Raspberry Pi 5 running 10+ containerised services, secured behind Tailscale VPN and Cloudflare Tunnels, with autonomous management via Claude Code and nightly AI-powered security audits.

Architecture

01
Services
Docker Containers

Home Assistant (smart home hub), n8n (workflow automation), Pi-hole (DNS ad blocking), Jellyfin (media server), Nextcloud (file storage), Syncthing (file sync), Glances (system monitoring), and others.

Systemd Services

nginx (reverse proxy with SSL for all subdomains), Tailscale VPN, Cloudflare Tunnel, Raspotify and Shairport-sync (Spotify Connect and AirPlay).

02
Network and Security
  • SSH restricted to Tailscale IP and localhost only. Key-only authentication, single allowed user.
  • Custom iptables rules: ts-input chain whitelists 5 authorised Tailscale peers, drops all other Tailscale traffic, blocks spoofed CGNAT addresses.
  • Wildcard SSL certificates for *.kempsonseaton.com via Let's Encrypt with automated renewal.
  • Tailscale mesh VPN for remote access. Tailscale Funnel disabled — nothing exposed via Tailscale serve.
03
Autonomous Management via Claude Code

CLAUDE.md defines operating rules: read memory on session start, run a drift check (Docker status, failed services, disk usage), never delete or restart without explicit confirmation.

An Obsidian vault stores structured state across three files: memory.md (hardware, OS, services, network, known issues, session log), network.md (interfaces, port map, DNS, peers, firewall rules, VPN routing), and projects.md (active, planned, and completed projects). Vault synced every 10 minutes via cron.

04
Nightly Security Audit

A cron job at 3am runs Claude Opus for a comprehensive audit covering: port exposure, firewall chain validation (IPv4 and IPv6), VPN routing assessment, SSH auth log review, Docker privilege review, Tailscale peer verification, SUID binary check, world-writable file scan, cron job audit, and system integrity checks.

Outputs a structured markdown report with a risk level (GREEN / AMBER / RED), critical issues, warnings, and prioritised recommended actions. Reports are stored in the Obsidian vault with a README index.

Technical Decisions

Why a Raspberry Pi

Low power (~£5/month electricity), always-on, quiet, and physically located at home for LAN access to IoT devices. Sufficient for the workload.

Claude Code as sysadmin

Not replacing manual administration. Acts as a structured memory and drift-detection layer. Every session reads the current state before making changes. Prevents configuration drift across dozens of services.

Obsidian vault over a database

Plain markdown files are human-readable, git-trackable, and editable from any device. No database dependency to fail.
