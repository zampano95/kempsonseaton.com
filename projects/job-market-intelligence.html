Job Market Intelligence Pipeline

A daily search across UK job boards for AI automation roles. Claude scores each listing against a detailed professional profile and groups them by realistic fit. The output is a colour-coded HTML email with application advice and a skills gap radar.

Architecture

Five-stage pipeline: Search → Analyse → Skills Intelligence → Format → Distribute.

01
Job Market Search

Adzuna API (UK job aggregator) queried across 10 search terms: AI automation, LLM engineer, n8n automation, prompt engineer, AI consultant, workflow automation, AI operations, AI agent developer, and others.

Each query returns up to 20 results filtered to the last 3 days. Results are deduplicated by job ID across queries. Typical daily yield: 100 to 150 unique listings.

02
AI Profile Matching (Claude Sonnet)

All listings are sent with a detailed professional profile. Claude performs five analyses per job:

  • Relevance score (1 to 5): alignment with AI automation career direction
  • Fit score (1 to 5): honest assessment against current experience
  • Categorisation: strong_match / worth_applying / stretch / out_of_reach
  • Work mode extraction: remote / hybrid / office / unknown. Non-London, non-remote roles are filtered out.
  • Application advice: which specific experience to emphasise
03
Skills Intelligence (Claude Haiku)

Extracts skill frequency across all matched listings and cross-references against the candidate profile. Each skill gets a traffic-light status: green (have it), amber (partial), red (gap).

Produces three prioritised learning recommendations based on high-frequency gaps. The job market becomes a personalised curriculum recommendation engine rather than a passive digest.

04
Email Formatting

Colour-coded HTML email: green (strong match), yellow (worth applying), orange (stretch), red (out of reach). Each listing shows title, company, location, work mode, salary, scores, reasoning, and specific application advice.

A Skills Radar section follows with a frequency table and the three prioritised learning recommendations.

05
Distribution

Gmail SMTP. Delivered at 7:30am daily.

Technical Decisions

Adzuna over LinkedIn

LinkedIn blocks scraping. Adzuna provides a free API that aggregates thousands of UK job boards, giving broader coverage without access restrictions.

Wide net + AI filter over keyword matching

10 broad search queries cast a wide net; Claude filters the false positives. This catches non-standard job titles that keyword filtering would miss. "Intelligent Process Automation Lead" won't appear in a search for "n8n automation", but Claude recognises the relevance.

Honest fit scoring

The prompt explicitly instructs Claude to be honest rather than optimistic. Out-of-reach roles are still surfaced and categorised clearly, so the digest builds an accurate picture of the market without encouraging wasted applications.

Defensive JSON parsing

Strips markdown backticks, locates array boundaries, handles truncated responses. Claude occasionally wraps JSON in markdown despite explicit instructions not to. Handling this in the workflow prevents silent failures in production.

Token budget and payload truncation

Initial implementation hit timeouts sending 131 listings in a single request. Solved by increasing max_tokens to 8192 and adding payload truncation to keep requests within a reliable size.

Infrastructure

  • Raspberry Pi 5 · Docker
  • n8n (self-hosted, open-source)
  • Claude Sonnet (profile matching)
  • Claude Haiku (skills analysis)

  • Cron daily, no manual intervention
